# Release This Version

With Appcircle's **Release This Version** feature, you can release your submitted version on **App Store Connect**. With this feature, you can easily release the relevant version to App Store without going to App Store Connect.

<Screenshot url='https://cdn.appcircle.io/docs/assets/CSM-261-release-version.png' />

:::caution Release This Version

This feature is only active for versions with a status of **Pending Developer Release** on App Store Connect. If the version does not have this status, the feature will appear as disabled.

<Screenshot url='https://cdn.appcircle.io/docs/assets/CSM-261-pending-dev-new.png' />

:::

When this feature is used, Appcircle will display a confirmation screen. This confirmation screen indicates that the relevant binary will be released directly to the App Store. When you click the `Release This Version` button, Appcircle automatically releases the relevant version to the App Store.

:::danger Release This Version

Please remember that once the app is released to the App Store, the release cannot be undone.

:::

<Screenshot url='https://cdn.appcircle.io/docs/assets/CSM-261-pop-up.png' />


Once the release process is successfully completed, the store status for the relevant version will also be updated according to the App Store Connect status. When the release process is complete, the status will be updated to `Ready for Sale`.

<Screenshot url='https://cdn.appcircle.io/docs/assets/CSM-261-ready.png' />
